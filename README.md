# Found in Space Website

Astro site for `foundin.space`, an educational platform built around Gaia-based data of the solar neighbourhood.

This repository is intentionally starting with content architecture before visual design and page production.

## Current focus

- define the site's purpose and audiences
- map the information architecture
- establish reusable content types for learning, exploration, and technical documentation
- prepare for later implementation in Astro

## Planning docs

- `docs/information-architecture.md`
- `docs/content-model.md`

## Development

```sh
npm install
npm run dev
```

## Initial site shape

The working IA currently assumes five primary sections:

- `Home`
- `Explore`
- `Learn`
- `Build`
- `About`

An `Updates` section is optional and can be added once the project begins publishing regular notes, releases, or teaching pilots.
